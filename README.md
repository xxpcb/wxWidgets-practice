# wxWidgets-practice
wxWidgets+wxFormBuilder+OpenCV

## 测试环境
* wxWidgets 3.0.3
* wxFormBuilder 3.5.1
* OpenCV 3.3
* Visual Studio 2015 Community

## 文件结构说明
* common 存放所有例程共用的函数文件
* img 存放测试图片
* 其余的文件夹为单独的例程代码（包含代码与fpb文件）

## 例程目录

* #### calcSquare
  计算平方数

* #### cameraTest
  摄像头测试

* #### findContours
  查找图片轮廓

* #### openPic
  打开图片测试
